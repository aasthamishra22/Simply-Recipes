**Indian Vegetarian Recipe Website**

A simple and inviting recipe website showcasing traditional and modern Indian vegetarian dishes. Built with **MERN stack** (MongoDB, Express, React, Node.js) and styled using **Tailwind CSS** + **Google Fonts** for a clean, accessible design.

Live Demo
Check out the deployed site here: https://reliable-pie-29e180.netlify.app/

---

## Features
- Browse curated Indian vegetarian recipes  
- Step-by-step cooking instructions  
- Ingredient lists with measurements  
- Responsive design for mobile & desktop  
- Warm, accessible UI with custom fonts
